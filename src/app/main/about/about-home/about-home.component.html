¿Qué es FlatMate?


FlatMate es una plataforma de búsqueda de alojamiento para todo tipo de personas, a las cuales queremos facilitarles el proceso de busqueda de un apartamento o una habitación para alquilar con total confianza.

Nace como idea de un proyecto de Máster, tras el que con mucho esfuerzo de sus integrantes, termina en un servicio con varias funcionalidades para la muestra de habitaciones y apartamentos disponibles para alquilar. Queremos que cualquier persona se sienta segura al momento de realizar una reserva, ofreciendo un excelente servicio. Esta es nuestra manera de revolucionar un sector anticuado. Esta es nuestra forma de empezar con un nuevo desafio, aunque este el solo el primer paso para le desarrollo de un proyecto de mejor calidad.

Flatmate es un proyecto de Máster, con el cuál podrás encontrar alojamiento y/o compañeros de piso en Santiago de Compostela, Coruña, Lugo Vigo Pontevedra y Ourense. Ayúdanos a que otras personas como tú encuentren a su compañero ideal!!
